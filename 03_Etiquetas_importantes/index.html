Etiquetas Mas Importantes y Utilizadas

Layout

<header> Se utiliza englobar todo el encabezado de mi pagina </header>
<nav> Se utiliza para poner los menu de navegacion </nav>
<section> Se utiliza para separar el contenido en secciones unicas </section>
<footer> Se utiliza para el pie de pagina de nuestra web </footer>
<aside> Se utiliza para poner elementos aparte del contexto </aside>
<article> Se utiliza para separar elementos diferentes dentro de las secciones </article>

Enlaces

<a href=""> Se utiliza como enlaze para navegar en la web </a>

Texto

<h1> Se utiliza para el titulo de mi web, solo debo tener una vez esta etiqueta, es el encabezado mas importante </h1>

<h2> Se utiliza igual para encabezado pero menos importantes que h1, se puede repetir </h2>

<h3> Se utiliza igual para encabezado pero menos importantes que h2, se puede repetir </h3>

.....

<h6> Se utiliza igual para encabezado pero es el menos importante </h6>

<p> Se utiliza para escribir contenido, es la mas utilizada para el texto, todo contenido dentro de un articulo o seccion se rellena dentro de esta etiquetea de parrafo </p>
