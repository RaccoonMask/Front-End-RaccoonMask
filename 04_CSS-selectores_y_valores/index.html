Selectores Basicos

De tipo

div {...} selectores que se aplican a las etiquetas HTML se aplican a todas las etiquetas

De Clase

.nameclass {...} selector de clase se aplica a elementos especificos es el mas utilizado ya que es muy selectivo y flexible

De ID

#id {...} selectores de ID no se deben utilizar ya que el ID es unico y se utiliza en JavaScript como referencia

De Atributo

a[href=".."] {...} Selector de atributo se aplican depende de su atributo por ejemplo de href unico asi aplica a ese solo

Universal

* {...} selector universal, es el que se utiliza para aplicar a todas las etiquetas de la web

Ejemplos

  • Blanco
  • Google
  • Ogame
